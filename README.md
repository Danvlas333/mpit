# Jarvis Study

Локальное веб-приложение для учебы: домашние задания, напоминания, архив, успеваемость ученика и чат-помощник.

## Возможности

- вход по логину и паролю;
- planner с домашними заданиями и личными напоминаниями;
- архив выполненных и пропущенных задач;
- страница успеваемости с оценками по четвертям;
- чат с ИИ через Ollama;
- голосовой синтез речи через Piper.

## Требования

- Python 3.11+;
- установленный [Ollama](https://ollama.com/) и доступная модель;
- локальные голосовые модели Piper в папке `mpit/`.

Python-зависимости ставятся из `requirements.txt`, но есть и внешние компоненты, которые через `pip` не ставятся:

- установленный и запущенный Ollama;
- скачанная модель Ollama, например `qwen2.5:3b`;
- `.onnx`-файлы голосов Piper в папке `mpit/`.

## Установка

1. Создайте и активируйте виртуальное окружение:

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
```

2. Установите зависимости:

```powershell
pip install -r requirements.txt
```

3. Установите или проверьте внешние компоненты:

- Ollama должен быть установлен отдельно;
- модель для чата должна быть заранее скачана;
- голосовые модели Piper должны лежать локально.

Пример для Ollama:

```powershell
ollama pull qwen2.5:3b
```

4. Убедитесь, что Ollama запущен, и при необходимости задайте модель:

```powershell
$env:OLLAMA_MODEL="qwen2.5:3b"
```

## Запуск

```powershell
python app.py
```

По умолчанию сервер стартует на:

```text
http://127.0.0.1:8000
```

При необходимости можно переопределить адрес и порт:

```powershell
$env:HOST="127.0.0.1"
$env:PORT="8000"
python app.py
```

## Вход в систему

После первого запуска база создаётся автоматически. Доступны тестовые пользователи:

- `student` / `student123`
- `teacher` / `teacher123`

## Как пользоваться

### 1. Вход

- откройте `http://127.0.0.1:8000`;
- введите логин и пароль;
- после входа откроется dashboard.

### 2. Главный экран

- слева находятся кнопки навигации;
- в центре показываются домашние задания и напоминания;
- сверху можно выбрать дату;
- если выбрать дату, показываются задания только на этот день;
- если нажать на домик, показываются все задания и напоминания.

### 3. Чат

- внизу находится поле ввода;
- можно писать вопрос или задачу в свободной форме;
- учебные вопросы уходят в чат;
- задачи и напоминания сохраняются в planner.

Примеры:

- `Объясни закон Ома простыми словами`
- `Напомни купить тетрадь 25 апреля`
- `Помоги с английским`

### 4. Архив

- выполненные задачи попадают в архив;
- просроченные задачи и домашние задания попадают в раздел пропущенных;
- удалённые элементы попадают в корзину.

### 5. Успеваемость

- страница оценок показывает четверти, средние оценки и диаграмму;
- стрелками можно переключать четверти;
- слабый предмет определяется автоматически по средней оценке;
- кнопка с самолётиком переносит в чат с готовым вопросом по самому слабому предмету.

## Где хранятся данные

- пользователи и сессии: `jarvis_study.db`
- пользовательские данные: `user_data/<storage_key>/`
- шаблоны интерфейса: `templates/`

Основные пользовательские файлы:

- `state.json` — домашние задания, заметки и история planner;
- `completed_tasks.json` — архив;
- `student_performance.json` — экран успеваемости по урокам;
- `student_grades.json` — оценки и аналитика по четвертям;
- `chats/*.json` — отдельные чаты.

## Основные файлы проекта

- `app.py` — HTTP-сервер и API;
- `auth.py` — авторизация и сессии;
- `planner.py` — логика обработки учебных запросов и задач;
- `user_store.py` — хранение пользовательских данных;
- `tts_engine.py` — синтез речи.

## Примечания

- для работы чата должен быть доступен Ollama;
- для работы TTS нужен установленный Python-пакет Piper и `.onnx`-модели голоса;
- если интерфейс показывает старые данные, обновите страницу с очисткой кэша.
